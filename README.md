 goVihara

🌍 Overview

goVihara is a full-stack travel discovery and tourism platform designed to help users explore destinations across India. The platform enables travelers to discover attractions, access travel recommendations, plan budgets, view weather information, and explore destinations through an intuitive interface.

✨ Features

* Destination Discovery
* Tourist Attraction Recommendations
* Travel Budget Calculator
* Weather Information Widget
* Travel Recommendation Engine
* Interactive User Interface
* State-wise and City-wise Travel Exploration

🛠️ Tech Stack

Frontend

* Reflex (Python Web Framework)

Backend

* Supabase

Database

* PostgreSQL

Development

* Python
* Git
* GitHub

📂 Project Structure

* `govihara_.py` – Main Application
* `destination_card.py` – Destination Display Component
* `attraction_card.py` – Attraction Information Component
* `recommendation_card.py` – Recommendation Engine UI
* `budget_calculator.py` – Travel Budget Planning Module
* `weather_widget.py` – Weather Information Component
* `navbar.py` – Navigation Component
* `footer.py` – Footer Component

🚀 Future Enhancements

* AI-Powered Itinerary Generator
* Personalized Travel Recommendations
* Real-Time Maps Integration
* User Reviews and Ratings
* Advanced Destination Search
* Smart Budget Prediction
* Multi-Day Travel Planning

🎯 Purpose

The goal of goVihara is to simplify travel planning and destination discovery by providing travelers with a centralized platform for exploring India's tourism ecosystem.
